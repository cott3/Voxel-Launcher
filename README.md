# Voxel Launcher

The **Voxel Launcher** is a JavaScript-based minimal launcher for Minecraft. The project is still in development, so there may be some bugs.

---

## JAVA

Launcher curently supports all version of java from **https://adoptium.net/**
other java platform support comeing soon.

---

## Installation

You can download from rleases and install it using the installer.

The launcher currently **only supports Windows**,cross platform support comeing soon for Linux.

---

## NPM and Node

The launcher requires **Node 18+**. If you are cloning the project, you need to install npm at the root of the project.
